### Bugs fixed (this session)
1. **Hyphen in command regex** — `/\w+/` doesn't match `-`, so `/task-today` captured only `task`. Fixed: `/^\/([\w-]+)/`
2. **Chat ID type mismatch** — `ALLOWED_CHAT_IDS` had strings `['8483054608']` but Telegram sends numbers. Fixed: `[8483054608]` (number)
3. **Removed `parse_mode: 'Markdown'`** — `run_task_cmd.py` output has mixed formatting that Telegram's Markdown parser chokes on; sends as plain text now
