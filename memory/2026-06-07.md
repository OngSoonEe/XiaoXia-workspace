### Browser/Agoda Lessons Learned
- Agoda search is extremely anti-bot — direct search rarely works
- Autocomplete is the most reliable way to find specific hotels on Agoda
- URL guessing for Agoda hotel slugs fails (404)
- Trip.com also uses whaleguard anti-bot that blocks detail pages
- Bing/DuckDuckGo search for Agoda links yields poor results
- Best approach: use Agoda autocomplete, then select from dropdown
