### Critical Lessons
1. **`reveal` class + IntersectionObserver** can silently hide elements forever if the observer doesn't fire. When in doubt, remove `reveal` from critical above-the-fold content.
2. **SVG and CSS percentage positioning don't mix** — viewBox coordinates (0-400) don't map to CSS percentage positions. Use pure CSS grid instead.
3. **"Same structure, different shapes" is NOT "different enough"** — Jack rejected the grid version because it still felt like Wolvenix (4 satellites around a center). Had to change the fundamental layout concept (pipeline vs orbit).
4. **Installed Playwright Chromium** for headless browser screenshots — useful for visual verification when no GUI browser is available. Screenshot saving via CDP `Page.captureScreenshot` with base64 → decode to file works reliably.
5. **Always verify on server** — checked FTP-served CSS to confirm uploads were live (not just local file changes).
